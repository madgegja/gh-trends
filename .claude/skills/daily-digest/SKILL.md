# Daily digest

Build today's GitHub trending digest and persist it.

## Task

1. Get today's date: !`date +%Y-%m-%d`
2. Use the `trend-researcher` subagent (via the Agent tool) THREE times in parallel:
   - overall daily: no language filter
   - python weekly
   - `$0` weekly (only if `$0` is non-empty)
3. Combine the three digests into a single markdown file with this header:
   ```
   # GitHub trending digest — <date>

   _Generated by daily-digest skill._
   ```
   Then sections: `## Overall (daily)`, `## Python (weekly)`, and `## $0 (weekly)` if applicable.
4. Write the result to `digests/<date>.md`. If the file already exists, append a `## Re-run @ HH:MM` section instead of overwriting.
5. Print a one-line summary to the user: file path + the top 3 themes across all three views.

ARGUMENTS: $ARGUMENTS
